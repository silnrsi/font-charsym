FONTLOG for the CharSym fonts

This file provides detailed information on the CharSym Font Software.
This information should be distributed along with the CharSym fonts (derived from the Bloom Show Inv fonts) and any derivative works.

Basic Font Information

CharSym is an open font that can be used by applications to show symbols
that represent 'invisible' characters (such as NBSP). Font sources are in UFO3 format.
